# Política de Segurança (Security Policy)

## Versões Suportadas

Atualmente, apenas a versão mais recente do projeto recebe atualizações de segurança.

| Versão | Suportada |
| ------ | --------- |
| < 1.0  | Sim       |

## Reportando uma Vulnerabilidade

Levamos a segurança deste projeto a sério. Se você descobrir uma vulnerabilidade de segurança, pedimos que **não** crie uma Issue pública. Em vez disso, siga o procedimento abaixo:

1. Envie um e-mail detalhado para [seguranca@vidasmasculinas.com.br](mailto:seguranca@vidasmasculinas.com.br).
2. Inclua o máximo de informações possível, como passos para reproduzir, impacto potencial e qualquer prova de conceito (PoC).
3. Entraremos em contato em até 48 horas para acusar o recebimento do reporte.
4. Manteremos você informado sobre o progresso da correção antes de sua divulgação pública.
